## Features

<CardGrid stagger>
	<Card title="Zero-config" icon="star">
		Support for Node, Python, Go, and PHP out of the box. (more coming soon!). First class support for Vite, Astro, and CRA static sites.
	</Card>

	<Card title="Build on BuildKit" icon="rocket">
		Efficient image construction using BuildKit LLB. Implemented as a custom frontend.
	</Card>

	<Card title="Configure your site" icon="setting">
		Customizable through environment variables or a config file.
	</Card>

	<Card title="Read the docs" icon="open-book">
		Learn more in [the Railpack Docs](https://starlight.astro.build/).
	</Card>
